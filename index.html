Comandos de Linux

1. cat index.html, muestra el contenido del archivo

2. sudo mv carpeta1 carpeta2, mover una carpeta dentro de otra

3. sudo brinda permisos de administrador

4. ls, lista los archivos

5. man cat, manual de cat

6. touch index.html, crear un archivo

7. mkdir, crear una carpeta

8. sudo apt-get update -> gestor/repositorio de dependencias, (sinónimo de Store). Mantiene actualizado el repositorio

9. apt-get update, si sale error entonces le damos permisos mediante sudo apt-get update

10. sudo apt-get upgrade

11. sudo apt install nodejs

12. sudo apt install npm - manejador de paquetes de Node

13. npx sirve para ejecutar paquetes sin necesidad de mantenerlos instalados, npx sirve para aquéllo que solo se va a ejecutar 1 vez. npm es para paquetes permanentes. npx, es descargar los paquetes y borrar el instalador.

14. npx create-react-app test, "primer proyecto en React.js"

15. sudo chown -R user ~/personalProjects

Git es el gestor de versiones. GitHub es una plataforma de subida de archivos (Gitlab también)

16. borrar archivo, rm nombreArchivo.extension
borrar una carpeta vacía, rm -d nombredelacarpetavacia
borrar carpeta con contenido dentro, rm -rf test
rm -rf test

close folder en VSCODE para cerrar un proyecto vacío
